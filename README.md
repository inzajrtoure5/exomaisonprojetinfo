\# Exercices Maison - Projet Informatique



\## Contenu



\### Exercice 1 - Gestionnaire de Contacts

Programme Java qui gère une liste de contacts avec ajout et recherche par nom.



\### Exercice 2 - Jeu de Devinettes

Jeu en Java où l'utilisateur doit deviner un nombre aléatoire entre 1 et 100 en 10 essais.



\### Exercice 3 - Nombre de Mots

Programme Java qui lit un fichier texte et compte le nombre total de mots.



\## Prérequis

\- Java JDK 26

\- IntelliJ IDEA
