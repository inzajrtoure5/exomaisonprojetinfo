# Nombre de Mots

## Description
Programme Java qui lit un fichier texte et compte le nombre total de mots.

## Fonctionnement
- Lit chaque ligne du fichier avec un File Scanner
- Sépare chaque ligne avec l'espace comme délimiteur
- Compte les mots ligne par ligne
- Affiche le nombre total de mots

## Comment exécuter le projet
1. Cloner le dépôt :
```bash
git clone https://github.com/inzajrtoure5/exomaisonprojetinfo.git
```
2. Ouvrir dans IntelliJ IDEA
3. Lancer `Main.java`

## Prérequis
- Java JDK 21
- IntelliJ IDEA
