# Jeu de Devinettes – Java

## Description
Un jeu en ligne de commande où l'utilisateur doit deviner un nombre
aléatoire entre 1 et 100.

## Règles
- Le programme génère un nombre secret entre 1 et 100
- L'utilisateur a **10 essais** maximum
- Après chaque essai : trop grand ou trop petit
- Bon nombre trouvé → **victoire**
- 10 essais épuisés → **défaite**

## Comment lancer le projet
1. Cloner le dépôt :
```bash
   git clone https://github.com/inzajrtoure5/exomaisonprojetinfo.git
```
2. Ouvrir le projet dans IntelliJ IDEA
3. Lancer `Main.java`

## Prérequis
- Java JDK 21
- IntelliJ IDEA
